# 战备数据查询插件

AstrBot 战备数据查询插件，用于查询战备推荐方案。

## 功能

- 查询战备数据，支持地图 1-5
- 显示推荐方案详情，包括武器、装备、附件等信息
- 显示价格和价值信息

## 使用方法

发送指令：`/战备 [金额W]`

支持金额：
- `11W` - 零号大坝/长弓溪谷-机密 零号大坝-水淹 (11.25w)
- `18W` - 巴克什/航天基地-机密 零号大坝-永夜 (18.75w)
- `24W` - 潮汐监狱-适应 (24.75w)
- `55W` - 巴克什-绝密 (55w)
- `60W` - 航天基地-绝密 (60w)
- `78W` - 潮汐监狱-绝密 (78w)

例如：
- `/战备 11W` - 查询 11W 战备值的推荐方案
- `/战备 18W` - 查询 18W 战备值的推荐方案
